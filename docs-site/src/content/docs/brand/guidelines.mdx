# Brand Guidelines

ThumbCode uses the P3 **"Warm Technical"** brand identity — technical but approachable, like a trusted tool that's also a friend.

## Visual Style: Organic Paint Daubes

ThumbCode uses **organic, imperfect shapes** — NOT gradients. This is critical brand differentiation.

### Do

- Use asymmetric border-radius
- Add subtle rotation to cards
- Use multi-layered organic shadows
- Apply SVG turbulence filters

### Don't

- Use linear or radial gradients
- Use perfectly rounded corners
- Use flat, sterile UI patterns
- Copy Material Design literally

### CSS Patterns

```css
/* Organic button shape */
.btn-organic {
  border-radius: 50px 45px 50px 48px / 26px 28px 26px 24px;
}

/* Organic card with tilt */
.card-organic {
  border-radius: 20px 18px 22px 16px / 16px 20px 18px 22px;
  transform: rotate(-0.3deg);
}
```

## Logo

The ThumbCode logo features a stylized thumbprint in Coral with organic paint texture.

### Usage Rules

- Maintain clear space equal to the logo height
- Never stretch or distort the logo
- Use the light version on dark backgrounds
- Use the dark version on light backgrounds

## Voice & Tone

ThumbCode communication should be:

- **Technical but accessible** — Use clear language, explain jargon
- **Confident but humble** — We're proud of our work, not arrogant
- **Helpful but not hand-holding** — Guide users, respect their intelligence
- **Warm but professional** — Friendly without being casual
